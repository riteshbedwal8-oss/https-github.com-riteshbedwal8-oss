### 📦 [Gems Quality & Pricing Analysis Dashboard](https://github.com/riteshbedwal8-oss/Gems-Quality-and-Pricing-Analysis-Dashboard)

An interactive Power BI dashboard analyzing gem quality metrics and pricing patterns. Provides visual insights into quality grades, price distribution, and market trends across different gem categories.

---

### 🏙️ [NYC 311 Customer Service Request Analysis](https://github.com/riteshbedwal8-oss/Customer-Service-Request-Analysis)
> **Python** | EDA | Statistical Analysis | Jupyter Notebook

A comprehensive analysis of **360,000+ NYC 311 service requests** using Python. Applied data wrangling, EDA, and statistical testing to uncover complaint patterns and response time insights.

- 📌 Engineered `Request_Closing_Time` feature to measure departmental response efficiency
- 📌 Identified **top 10 complaint types** and geographic hotspots using visualizations
- 📌 Applied **Kruskal-Wallis Test** — rejected H₀, confirming significant variation in response times across complaint types (p < 0.05)
- 📌 Performed cross-tabulation of complaint types across New York boroughs
